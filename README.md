## Install Server
- Move path
```shell
$ cd server
```
- Install package dependencies
```shell
$ yarn install # npm install
```

- Run server
```shell
$ yarn start # npm run start
```

## Install App
- Clone the project
```shell
$ git clone https://github.com/VMinhHa/yema-ecommerce.git
```
- Install package dependencies
```shell
$ yarn install # npm install
```
- Run app
```shell
$ yarn start # npm run start
